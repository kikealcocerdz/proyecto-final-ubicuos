Añadiendo productos a la lista de compra

Para añadir productos a su lista de compras debes pulsar el "plus" () blanco en la pantalla de empezar compra. Cada producto tiene su propio código, y cuando lo pulsas puedes decir "añadir" y el código del producto (por ejemplo, para añadir las tomates puedes decir "añadir TO24").

Eliminando productos de la lista de compra

Para eliminar un producto solo tienes que girar tu dispositivo móvil 180 grados mientras tocas el producto que quieres eliminar.

Marcando productos como favorito

Para marcar un producto como favorito, pulsa la estrella vacía que está en el producto y decir "favorito". (). Para eliminar un producto de favoritos pulsa la estrella completado y decir "favorito"()

Para usar la mapa

En la pagina de la mapa puedes ver las locaciones de los productos en la lista de compras.

Para pagar

Cuando has termiado la compra debes tocar para navigar a la pagina final. Debes buscar a la persona responsable en caja que tiene un codigo QR que peudes escanear para pagar.
