Añadiendo productos a la lista de compra

Para añadir productos a su lista de compras debes pulsar el "plus" blanco en la pantalla de empezar compra. Cada producto tiene su propio código, y cuando lo pulsas puedes decir "añadir" y el código del producto (por ejemplo, para añadir las tomates puedes decir "añadir ABC12").

Eliminando productos de la lista de compra

Para eliminar un producto solo tienes que girar tu dispositivo móvil 180 grados mientras tocas el producto que quieres eliminar.

Marcando productos como favorito

Para marcar un producto como favorito, pulsa la estrella que está en el producto.
